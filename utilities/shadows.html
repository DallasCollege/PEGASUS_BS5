Shadows can be applied on objects in PEGASUS with three different depths, .shadow-1, .shadow-2 and .shadow-3.

Shadow 1

This is a simple div with shadow-1 applied. 
<div class="shadow-1">
    This ... shadow-1 applied.
</div>

Shadow 2

This is a simple div with shadow-2 applied. 
<div class="shadow-2">
    This ... shadow-2 applied.
</div>

Shadow 3

This is a simple div with shadow-3 applied. 
<div class="shadow-2">
    This... with shadow-3 applied.
</div>

Proper Use: Shadows

Use these classes when you want to add visual importance to a button, box or <div>
