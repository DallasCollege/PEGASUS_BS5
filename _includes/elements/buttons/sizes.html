Button Sizes

In addition to the default button shown above, there are four additional button sizes available.

Add any of the size classes below to <a class="button">:

  • .button-xl - makes button text: font-size: 1.5rem /* 24px */
  • .button-lg - makes button text: font-size: 1.25rem /* 20px */
  • .button-sm - makes button text: font-size: .8625rem /* 14px */
  • .button-mn - makes button text: font-size: .625rem /* 10px */; text-transform: uppercase
