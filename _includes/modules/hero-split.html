Hero Split Section

Only for full width page layouts. Not for pages with sidebars.

Hero Split Sections allow for large hero images and text to wrap across the entire screen, outside of the page container. This component includes:

  • an area for text/content
  • an area for a responsive hero image

Basic Code and Classes:

<div class="full-width-row">
    <section class="hero-split-section cprimary-lt">
        <div class="col-sm-6 hero-split-photo"></div>
        <div class="container">
            <div class="row">
                <div class="col-sm-6 col-sm-offset-6 hero-split-content">
                    <!-- Content starts here -->
                        <h2 class="cb-title-xl">Employee Success</h2>
                        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed veniam.</p>
                        <a href="#" class="button cprimary">Button</a>
                    <!-- Content ends here -->
                </div>
            </div>
        </div>
    </section>
</div>
  • Use the bootstrap grid column classes to control where the "split" occurs across this width of the page. This example uses .col-sm-6 for the width of the photo, and .col-sm-6 for the width of the content area.
  • You can put any HTML content inside .hero-split-content

Style This Component

Notice how variations to the layout above can be made by customizing individual PEGASUS elements like the following:

  • The color of the background of the text (.bg-cprimary-lt is used in this example)
  • The size and style of the heading (h2.cb-title-xl is used in this example)
  • The style, color or size of button (.button-md.bg-cprimary-lt is used in this example)
