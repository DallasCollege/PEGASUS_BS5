Sidebar navigation (and some banner navigation) are populated from each subsite's Navigation Settings.

Access Navigation Settings

  1. Go to Site Settings
  2. Click Navigation under the "Look and Feel" heading

Setting Up Navigation for New Subsite

  • Ignore "Global Navigation" options
  • Under "Current Navigation" select the 2nd Structural Navigation option (Display only the navigation items below the current site)
  • Under "Structural Navigation: Sorting" select "Sort manually"
  • Under "Structural Navigation: Editing and Sorting" you will need to Add a link for the default page of the subsite
  • Input the URL of the default page as well as the title of the Subsite; leave Audience and Description blank
  • Make sure to move the added link to the very top of the list
  • The rest of the pages can be rearranged if need be
